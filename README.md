# 外贸报价单发票生成器

一个离线使用的外贸报价单、形式发票、商业发票、装箱单和利润核算工具。项目不依赖服务器，数据保存在当前浏览器本地。

## 功能

- 生成报价单、形式发票、商业发票和装箱单
- 支持产品图片、Logo、贸易条款、银行信息和备注
- 支持打印或另存为 PDF
- 支持导出 Excel
- 包含利润核算、包装数据管理和物流费用管理

## 使用方法

### 下载

1. 打开本项目的 GitHub 页面。
2. 点击页面右上方绿色的 `Code` 按钮。
3. 点击 `Download ZIP`。
4. 下载完成后，右键压缩包，选择“全部解压”或“解压到当前文件夹”。

### Windows 启动

1. 打开解压后的文件夹。
2. 再打开里面的 `报价单生成器` 文件夹。
3. 双击 `QuotationGenerator.cmd`。
4. 如果 Windows 弹出安全提示，选择“更多信息”，再选择“仍要运行”。

如果无法启动，请确认电脑已安装 Chrome 或 Microsoft Edge。

请不要把 `报价单生成器/index.html` 这段文字复制到浏览器地址栏。它是项目里的文件位置，不是网页链接。

## 隐私说明

本工具默认不上传数据，也不连接外部接口。填写的公司、客户、报价、银行和物流信息会保存在当前浏览器的本地存储中。分享给他人前，请不要把自己浏览器里的导出文件、截图或已填写资料一起打包。

开源版本已清理默认公司信息、Logo 和银行信息。使用者需要自行填写自己的资料。

## 开发说明

这是一个纯前端项目，不需要安装依赖即可运行。主要文件：

- `报价单生成器/index.html`：单据生成器
- `报价单生成器/app.js`：单据生成逻辑
- `报价单生成器/calculator.html`：利润核算器
- `报价单生成器/calculator.js`：利润核算逻辑
- `报价单生成器/packaging.html`、`报价单生成器/packaging.js`：包装数据管理
- `报价单生成器/logistics.html`、`报价单生成器/logistics.js`：物流费用管理
